# LightBurn Web

## Features
- Comprehensive control for your laser engraver.
- User-friendly interface.
- Real-time monitoring of engraving processes.
- Support for various laser engravers.
- Built-in file management for easy project tracking.

## Architecture
LightBurn Web is designed using a modern web architecture, featuring:
- **Frontend**: Built with React for a dynamic user experience.
- **Backend**: Node.js and Express for robust server-side operations.
- **Database**: MongoDB for scalable data storage.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hazzy010/lightburn-web.git
   cd lightburn-web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the application:
   ```bash
   npm start
   ```

## Usage
- Open your web browser and navigate to `http://localhost:3000` to access the application.
- Follow the on-screen instructions to set up your laser engraver.
- Explore the features through the intuitive menu.
